# Baremetrics

Baremetrics is a subscription analytics platform providing a REST API for accessing MRR, ARR, churn rate, LTV, customer metrics, and revenue forecasting data from Stripe and other billing sources.

**API Base URL:** `https://api.baremetrics.com/v1`

**Developer Docs:** https://developers.baremetrics.com

**Status:** https://status.baremetrics.com
